# MCP45HVX1 v1.0.0

Initial release of the MCP45HVX1 high-voltage I2C digital potentiometer driver.

## Included

- Transport-agnostic core driver with injected I2C callbacks
- Volatile Wiper 0 read/write/increment/decrement API
- TCON0 terminal-control helpers
- Direct register access for documented registers
- General Call write/increment/decrement helpers
- Driver health tracking and recovery probe
- Native Unity tests and ESP32 PlatformIO example CLI
